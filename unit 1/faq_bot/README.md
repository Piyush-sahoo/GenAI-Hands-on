# Automated FAQ Bot (Unit 1)

This project implements idea #17: **Automated FAQ Bot** using a Hugging Face extractive QA pipeline.

## What it does

- Loads a policy document from context.txt.
- Answers user questions by extracting spans from the policy.

## Requirements

- transformers
- torch

## Usage

### Option A: Notebook (recommended)

1. Open FAQ_Bot.ipynb.
2. Run the cells in order.
3. (Optional) Uncomment the install cell if needed.


## Customize

- Edit context.txt to replace the policy with your own.
